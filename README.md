#  MineCards

**MineCards** — это захватывающая карточная гача-игра, интегрированная прямо в Telegram (Mini App). Испытывайте удачу каждые 6 часов, собирайте уникальные карточки разной редкости.

---

## ✨ Основные возможности

* **🎰 Система Гача:** Получайте новую случайную карточку каждые 6 часов.
* **💎 Система редкостей:** От обычных до легендарных экземпляров
* **🎒 Инвентарь:** Удобный просмотр вашей коллекции в реальном времени.
* **📱 Telegram Native:** Плавная работа внутри интерфейса Telegram Mini Apps.
